# Global-Memory-Representation
This project examines how wartime atrocities associated with Germany and Japan are represented in global museum and educational narratives.

## Research Question
How are German and Japanese wartime atrocities represented differently in globsl educational and memorial institutions in terms of frequency, attribution of responsibility, and moral framing?

## Project Goal
The goal of this project is to explore how historical events become institutionalized in global memory through museums and educational matertials.

## Data Sources
- Museum educational texts
- Historical overview matericals from museum websites

## Method
This project uses Python-based text analysis to examine the frequency and framing of key terms related to responsibility, violence, and moral interpretation.

## Status
Early-stage research project focusing on building a dataset from museum narratives and conducting exploratory text analysis.
